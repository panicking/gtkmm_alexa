# Build

In order to build:

    git clone git@github.com:panicking/gtkmm_alexa.git
    cd gtkmm_alexa
    mkdir build
    cd build
    cmake ../
    make -j10
    ./src/alexa

Before starting the application be sure that Demo application is up and running
